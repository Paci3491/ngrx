With ChangeDetectionStrategy.OnPush, Angular will only check the component (and its subtree) when one of the following happens:

  1. An Input() reference changes (i.e., a new object/primitive value).
  2. An event is triggered inside the component. (Form input changes (click), (change), (keyup))
  3. An Observable emits (if used with async pipe in the template).
  4. You manually trigger change detection via ChangeDetectorRef.
